Tech Highlights

技術的なこだわり

🔒

OS レベルの
キーフック

Quartz CGEventTap で OS 全体のキーイベントをキャプチャ。どのアプリがフォーカスを持っていても動作

📡

PTY への
直接インジェクト

ioctl TIOCSTI を使い Claude Code の入力バッファに文字を直接書き込み。画面切り替え不要を実現

🎯

自PTY誤送信
防止機能

os.ttyname(0) でデーモン自身の PTY を検出・除外。正確に Claude Code のプロセスのみを対象にする

🖥️

ウィンドウ
フォーカス機能

F15 を押すと AppleScript で Claude Code のターミナルを自動フォーカス

🧵

ノンブロッキング
処理

キー送信・フォーカス操作を別スレッドで実行。イベントループをブロックしない

🚀

LaunchAgent
常駐起動

macOS LaunchAgent に登録することで Mac 起動時に自動スタート
